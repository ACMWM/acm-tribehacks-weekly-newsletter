Jobs and Opportunities

Google Summer of Code

Google Summer of Code gives students the opportunity to write code and learn about open source development while earning a stipend. 201 different open source organizations are participating this year! Applications are due April 3rd. Find out more here.


Google Code Jam

Google Code Jam returns for its 14th year as one of the most challanging programming competitions in the world. The contest culminates in the World Finals hosted at Google's Dublin office. Register online before April 7th. Find more details here.

The University of Maryland's annual hackathon, Bitcamp, is taking place April 7-9th. Bitcamp is definitely one of the largest MLH hackathons in the area - last year there were over 1000 participants! Register here.
